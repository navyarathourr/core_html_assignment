6. Create an ordered list of HTML tags. Each list item must include the tag name and some information about the tag.

Some HTML tags are:

  1. Comment

    2. The comment tag is used to insert comments in the HTML code.

  2. a href

    3. The anchor tag in HTML is used to create a hyperlink on the webpage.

  3. break

    4. The break tag inserts a single carriage return or breaks in the document. This element has no end tag.

  4. div

    5. The div tag is used in HTML to make divisions of content in the web page (text, images, header, footer, navigation bar, etc).

  5. footer

    6. The footer tag in HTML is used to define a footer of HTML document. This section contains the footer information.

  6. form

    7. This form is used basically for the registration process, logging into your profile on a website or creating your profile on a website, etc

  7. font

    8. The font tag in HTML plays an important role in the web page to create an attractive and readable web page.

  8. label

    9. The label tag in HTML is used to provide a usability improvement for mouse users.

  9. mark

    10. The mark tag in HTML is used to define the marked text. It is used to highlight the part of the text in a paragraph.

  10. nav

    11. The nav tag is used for declaring the navigational section in HTML documents. Websites typically have sections dedicated to navigational links, which enables users to navigate the site.

  11. table

    12. HTML Table, various ways to implement it, & will also understand its usage through the examples. HTML Table is an arrangement of data in rows and columns, or possibly in a more complex structure.

  12. strong

    13. The strong tag in HTML is the parsed tag and is used to show the importance of the text. Make that text bold.

  13. iframe

    14. The iframe tag defines a rectangular region within the document in which the browser can display a separate document

  14. button

    15. The button tag in HTML is used to define the clickable button.

  15. audio

    16. It is a useful tag if you want to add audio such as songs, interviews, etc. on your webpage.
