9. Create an ordered list of most common text formatting tags in HTML. Within each list item, use an unordered list to create the specific use cases and best practices for that tag.

  1. strong
    • The strong tag in HTML is the parsed tag and is used to show the importance of the text. Make that text bold.
    • This is strong tag.

  2. em
    • This tag is generally used to display a technical term, phrase, the important word in a different language. It gives emphasis to the specific word.
    • This is em tag.

  3. u
    • The underline tag in HTML stands for underline, and it's used to underline the text enclosed within the tag.
    • This is underline tag.

  4. mark
    • It is used to define the marked text. It is used to highlight the part of the text in a paragraph.
    • This is mark tag.

  5. strike
    • It is defined as a strike or line through Text.
    • This is strike tag.
